有关ID，有关阳光

Yifang Wang in ZJU 5月 23, 2017

我记得，小学，当同学们都在用稚嫩的语气描述出一个个美好、清丽的春天、都在用心赞美阳光的温暖的时候，我却总是喜欢一个人在阴天漫步，将自己装点成一个失意诗人，欲赋新词强说愁。那个时候不喜欢阳光，不喜欢一切太过炙热、美好的东西。喜欢阴沉、喜欢没有阳光的阴天，感觉那样的天气才是最好的，因为没有阳光的炙烤，所以不热；又没有雨水的浸润，所以不潮湿。多好。

那时候的自己不甘心于平凡、不甘心和别人一样，不甘心于一个个复制的生活。买书的时候，一定买书店存货最少的版本；穿衣服，从不穿裙子；也从来对打扮之类的事情嗤之以鼻；不留长发，一席短发清清爽爽；最喜欢的季节是冬天，雪白、肃杀、沉寂。

因为特立独行，所以勇敢的站在顶端。

可是，不知道从什么时候开始，自己已经完全不是原来的样子。我开始回归大众之中，甚至想要隐匿其中，不想要做那个拔尖、特殊的人。开始喜欢阳光、钦羡一切温暖的东西。春天，穿行在校园之中，看着大草坪上搭着帐篷来游玩的大人与孩子，看着一个个追着跑着叫嚷着的大小孩，看着一只只风筝飞翔在辽阔天空，旁边花红柳绿、水中飞禽嬉戏……心中是幸福的。嘴角不自觉地上扬，好像自己也已经沉浸其中了。

我渐渐意识到，一个人，可能在生活中越缺少什么，就越向往什么。孩童时期原本就充满快乐，所以想要体验所谓的“沉寂”与“肃杀”，想要感受那个时间感受不到的一份心情，所以远离阳光、远离人群，去找寻；而随着时间的流逝，我们经历着，经历更多的人和事，经历我们未曾想到的忧伤、难过和苦涩。我们经历了比儿时想要经历的更多更深的心情，这不是我们主动去追寻的，是生命安排给我们的历练和考验。我们避无所避，只能面对。我们开始疲累，开始思考生命中的各种色彩。然后发现，儿时最多的快乐几乎已经不见了，取而代之的，竟是那时候一直追求的阴郁和惆怅。可笑。发现自己当时的可笑。然而，人不一直是在追求永远得不到的，对已经拥有的从来都是不屑的吗？“珍惜”这个词，即使一些人在某个时间段做到了，但也未必长久。让人意识到拥有的东西的重要性的时刻永远都是失去它的瞬间。




成长的过程，也是认识生命、体验生命的过程，我们渐渐地接受心中美好的完美世界在自己的思想逐渐成熟的过程中慢慢崩塌，然后在某个点上停止崩塌。美好的和丑陋的，就这样，维持着动态平衡。

这绝不是一件坏事。这标志的，是一个 个体的真正的成熟。世界本该如此，我们现在做的，是让自己变得更加客观、理性，了解了世界的好与坏，我们才有可能去适应世界、去努力改变它，让它的那个好与坏的平衡点能够有机会往“好”的一段稍稍移动改一下。

这样，便已足够。

我骑行在林荫道上，茂盛的樟树已经长出了新叶。旧叶依旧在树枝上，不肯离去。这时候，我们需要做的，就是等待，等风来.

这是春天啊，暖暖的风，暖暖的光，暖暖的微笑……即使生命有太多支离与破碎，但若我们的心中一直在追寻那一份美好的绿意，那一切都能够过去。谁都无法将我们击垮。
